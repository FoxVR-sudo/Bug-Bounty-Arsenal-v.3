# Code of Conduct

## Our standard

This project expects respectful, constructive, and professional collaboration.

Examples of positive behavior:

- giving actionable technical feedback
- assuming good intent while discussing tradeoffs
- respecting different experience levels and backgrounds
- focusing criticism on code and behavior, not people

Examples of unacceptable behavior:

- harassment, intimidation, or personal attacks
- discriminatory or hateful language
- publishing private information without permission
- spamming, trolling, or intentionally derailing discussions

## Scope

This Code of Conduct applies to issues, pull requests, discussions, and other project communication spaces.

## Enforcement

Project maintainers may remove content, close discussions, or restrict participation when conduct violates these standards.
